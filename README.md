#  Objectif
Analyser les tendances du marché des cryptomonnaies (Top 200) afin de comprendre :
- l’évolution des prix (BTC, ETH, etc.)
- le volume échangé par crypto
- la capitalisation totale du marché



#  Données utilisées
- **Source :** [Kaggle – Cryptocurrency Prices (Top 200+)](https://www.kaggle.com/datasets/jessemostipak/cryptocurrency-prices-daily-updated)
- **Fichiers inclus :**
  - `crypto_clean_sample.csv` → version allégée (5000 lignes)
  - `crypto_prices_analysis.ipynb` → notebook d’analyse complet
  - `btc_evolution.png` → évolution du Bitcoin


# Analyses réalisées
- Nettoyage du dataset (dates, doublons, valeurs manquantes)
- Visualisation de l’évolution du Bitcoin et de l’Ethereum
- Analyse du volume total par crypto
- Étude de la capitalisation totale du marché crypto
