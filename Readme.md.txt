# Crypto Market Data Analysis

### Objectif
Analyser les tendances du marché des cryptomonnaies (Top 200) et comprendre les dynamiques de prix, volume et capitalisation.

### Dataset
Source : [Kaggle – Cryptocurrency Prices (Top 200+)](https://www.kaggle.com/datasets/jessemostipak/cryptocurrency-prices-daily-updated)

### Contenu du projet
- **crypto_clean.csv** → dataset nettoyé
- **crypto_prices_analysis.ipynb** → notebook complet
- **images/** → graphiques générés
  - btc_evolution.png
  - top10_volume.png
  - market_cap.png

### Résultats
- Le Bitcoin reste la crypto la plus échangée.
- Ethereum suit une dynamique parallèle, mais plus stable.
- La capitalisation globale montre des cycles haussiers marqués.
